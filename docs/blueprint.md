# **App Name**: PDFusion

## Core Features:

- PDF Upload: Select and upload multiple PDF files from the local file system.
- PDF Merge: Merge selected PDF files into a single PDF file. Allow users to specify the order of the merged PDFs.
- PDF Restructure: Rearrange pages from a PDF into one page, arrange horizontally or vertically. Let the user choose the number of rows and columns.

## Style Guidelines:

- Primary color: Light blue (#E3F2FD) for a clean and modern look.
- Secondary color: Gray (#607D8B) for text and subtle UI elements.
- Accent: Teal (#009688) for primary actions and highlights.
- Clean and intuitive layout with a focus on drag-and-drop functionality for file arrangement.
- Simple and clear icons for actions like 'Merge', 'Upload', and 'Download'.
- Subtle animations during file processing and merging to provide visual feedback.
